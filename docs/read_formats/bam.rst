Bam Outputs
===========

`Binary Alignment Map`_ (BAM) is the comprehensive raw data of genome sequencing; it consists of the lossless, compressed binary representation of the Sequence Alignment Map-files.

Bam files generated by minknow are compliant with the `Sam Specification`_, and can be manipulated using `Sam Tools`_.
